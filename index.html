PROGRAMMING LANGUAGES EXAMPLES
Procedural Programming Languages:
Emphasizes control flow and modular programming		 C, Pascal, Fortran
Object-Oriented Programming Languages:
Organizes code into objects, combining data and functions		 Java, Python, C++
Functional Programming Languages:
Focuses on immutability and avoiding side effects		 Haskell, Lisp, Scala
Markup Languages:
Used to define structure and presentation of documents		 HTML, XML, Markdown
Query Languages:
Used for querying and manipulating databases and structured data		 SQL, XQuery
Concurrency-Oriented Languages:
Designed for managing concurrent processes efficiently		 Erlang, Go
Domain-Specific Languages:
Designed for specific tasks or domains, like querying databases or formatting documents		 SQL, HTML, LaTeX
Compiled Languages:
Code is translated into machine code before execution		 C, C++, Rust
Interpreted Languages:
Code is executed line-by-line by an interpreter		 Python, JavaScript, Ruby
Scripting Languages:
Often used for web development, automation, and quick prototyping		 Python, JavaScript, Ruby
